tutor-contrib-shipyard plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/edunext/tutor-contrib-shipyard

Usage
-----

::

    tutor plugins enable tutor-contrib-shipyard
